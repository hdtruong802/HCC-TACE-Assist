# Antigravity workspace rule — Liver Cancer AI

Nguồn context chính của dự án là **`AGENTS.md`** ở thư mục gốc. Luôn đọc và tuân theo nó.

Trước khi làm:
1. Đọc `AGENTS.md` — scope, "Quyết định đã KHÓA", conventions bắt buộc (patient-level split, no leakage, threshold từ val).
2. Đọc **phần cuối** `docs/WORKLOG.md` để nối tiếp việc tool trước.
3. Chi tiết kỹ thuật: `report/T3_W1_Spec_Sheet.md`; kế hoạch: `docs/Liver_Cancer_AI_6Week_Plan.md`.

Đừng đọc lại toàn bộ repo — dùng Index trong `AGENTS.md`.

Kết thúc phiên: thêm entry lên đầu `docs/WORKLOG.md` + git commit.

> Nếu dùng Memory Bank của Antigravity: coi nó là bản sao tiện dụng, **không** phải nguồn sự thật. Nguồn sự thật = các file đã commit git ở trên.
